# BEM, css 방법론

## css 방법론

- 클래스들의 이름을 어떻게 체계적으로 지을 것인가?

## BEM

- `Block` + `Element` + `Modifier` 순으로 클래스 이름을 통일하자는 방법론

- `id`는 사용하지 않음.

### Block

- **재사용 가능한 페이지에 독립적인 컴포넌트 이름**

### Element

- **페이지에 의존적인 해당 페이지의 컴포넌트**

### Modifier

- **블럭이나 엘리먼트의 속성**

- clicke, hovered 이런 것들..
